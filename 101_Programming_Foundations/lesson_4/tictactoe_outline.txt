# Tic-Tac-Toe

1. Display the initial empty 3x3 board. (display_board)
2. Ask the user to mark a square. (user_marks_square)
3. Computer marks a square.  (computer_marks_square)
4. Display the updated board state. (update_board)
5. If winner, display winner. (winner?) (display_winner)
6. If board is full, display tie. (board_full) (display_tie)
7. If neither winner nor board is full, go to #2
8. Play again? 
9. If yes, go to #1
10. Good bye!
